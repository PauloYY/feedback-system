# Feedback System

## Descrição
Este projeto permite que usuários enviem feedbacks através de um formulário. 
O administrador pode visualizar, filtrar e analisar os feedbacks em um painel interativo com gráficos.

## Funcionalidades
- Envio de feedback pelo formulário
- Visualização de feedbacks no painel admin
- Filtro por categoria
- Marcar feedbacks como resolvidos
- Gráficos de análise de notas e categorias
- Autenticação de administrador (Firebase Auth)

## Tecnologias
- HTML, CSS, JavaScript
- Firebase (Firestore e Auth)
- Chart.js para gráficos
- Git/GitHub para versionamento

## Como rodar
1. Clone o repositório: https://github.com/PauloYY/feedback-system
2. Abra a pasta no VS Code
3. Abra o `index.html` no navegador
4. Configure o Firebase no arquivo `js/firebase/config.js`
